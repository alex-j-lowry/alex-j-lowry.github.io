Компьютеры СССР/Восточного блока

ZX-PK.ru - Вероятно, крупнейший форум по ретро-компьютерам в бывшем СССР. В основном посвящен Sinclair ZX Spectrum и его многочисленным клонам, но обсуждаются и другие советские и несоветские компьютеры.

nedoPC.org - Еще одно активное компьютерное сообщество бывшего СССР. Обсуждение многих тем, связанных с ретро-компьютерами.

Mick Laboratory - Информационный ресурс по различным компьютерам советской эпохи, включая перерисованные схемы и печатные платы.

ZXByte.ru - Информационный ресурс о клоне ZX Spectrum «Байт» из Беларуси. Также освещает другие советские компьютеры.

Специалист-ПК.рф - Информационный ресурс о самодельном компьютере «Специалист» на базе ВМ80 из СССР.

Специалист-ПК.рф (Раздел «Галаксия») - Информационный ресурс о самодельном компьютере «Галаксия» на базе Z80 из Югославии.

ex-YU Racunalniska Scena - Скачивайте и играйте в компьютерные игры из Югославии. Включает игры для компьютеров из Югославии и Запада. (На словенском языке)

Алексей Морозов - Информационный ресурс для многих советских компьютеров. У Алексея Морозова также есть канал на YouTube/RuTube.

Коллекция советской цифровой электроники - Обширный музей советской электроники.

Nostalcomp.cz - Информационный ресурс о компьютерах из Чехословакии. (На чешском языке)

Robotron Technik - Информационный ресурс по компьютерам из Восточной Германии. (На немецком языке)

Homecomputer DDR - Информационный ресурс по компьютерам из Восточной Германии. (На немецком языке)

CoBrasov.com - Информационный ресурс о клоне ZX Spectrum «CoBra» из Румынии. (На английском и румынском языках)

CoBrasov.org - Информационный ресурс о клоне ZX Spectrum «Turbo Spectrum» из Румынии. (На английском языке)

Bashkiria-2M.narod.ru - Информационный ресурс по компьютеру «Башкирия-2М» на базе ВМ80 из СССР.

Agat9.narod.ru - Информационный ресурс по серии компьютеров «Агат» на базе 6502 из СССР.

PK8000.narod.ru - Информационный ресурс по компьютерам «Сура», «Веста» и «Хобби» на базе ВМ80 из СССР.

PK8002.narod.ru - Информационный ресурс по компьютеру «Эльф» на базе ВМ80 из СССР.

PK8020.narod.ru - Информационный ресурс по компьютеру «Корвет» на базе ВМ80 из СССР.
