If you find the contents of this website interesting, I recommend checking out these other sites.

USSR/Eastern Bloc Computing

ZX-PK.ru - Probably the largest retro computing forum in the ex-USSR. Primarily dedicated to the Sinclair ZX Spectrum and its many clones, but other Soviet and non-Soviet computers are discussed too. (In Russian)

nedoPC.org - Another active ex-USSR computing community. Discussion of many retro computing topics. (Primarily in Russian, though there is an English section of the site)

Mick Laboratory - Information resource for various Soviet-era computers, including redrawn schematics and PCBs. (In Russian)

ZXByte.ru - Information resource for the "Byte" ZX Spectrum clone from Belarus. Also covers other Soviet computers. (In Russian, some pages translated into English)

Specialist-PK.rf - Information resource for the "Specialist" 8080-based homemade computer from the USSR. (In Russian)

Specialist-PK.rf (Galaksija Section) - Information resource for the "Galaksija" Z80-based homemade computer from Yugoslavia. (In English and Russian)

ex-YU Racunalniska Scena - Download and play computer games from Yugoslavia. Includes games for computers from Yugoslavia and the West. (In Slovenian)

Aleksey Morozov - Information resource for many Soviet computers. Aleksey Morozov also has a YouTube channel. (In Russian)

Soviet Digital Electronics Museum - An extensive museum of Soviet electronics. (In English and Russian)

Nostalcomp.cz - Information resource for computers from Czechoslovakia. (In Czech)

Robotron Technik - Information resource for computers from East Germany. (In German)

Homecomputer DDR - Information resource for computers from East Germany. (In German)

CoBrasov.com - Information resource for the "CoBra" ZX Spectrum clone from Romania. (In English and Romanian)

CoBrasov.org - Information resource for the "Turbo Spectrum" ZX Spectrum clone from Romania. (In English)

Bashkiria-2M.narod.ru - Information resource for the "Bashkiria-2M" 8080-based computer from the USSR. (In Russian)

Agat9.narod.ru - Information resource for the "Agat" 6502-based computer series from the USSR. (In Russian)

PK8000.narod.ru - Information resource for the "Sura", "Vesta" and "Hobby" 8080-based computers from the USSR. (In Russian)

PK8002.narod.ru - Information resource for the "Elf" 8080-based computer from the USSR. (In Russian)

PK8020.narod.ru - Information resource for the "Korvet" 8080-based computer from the USSR. (In Russian)
