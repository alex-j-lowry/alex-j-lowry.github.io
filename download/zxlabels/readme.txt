Because of GitHub's 25 MB file size limit, the GIMP XCF file for these images has been split up into several smaller files. You can use the Windows executable HJ-Split (included in this directory) to rejoin these files.

The fonts required for editing the XCF file can be found in fonts.zip.



Из-за ограничения GitHub на размер файла в 25 МБ, файл GIMP XCF для этих изображений был разбит на несколько меньших файлов. Вы можете использовать исполняемый файл HJ-Split для Windows (включенный в этот директорий) для объединения этих файлов.

Шрифты, необходимые для редактирования файла XCF, можно найти в файле fonts.zip.
